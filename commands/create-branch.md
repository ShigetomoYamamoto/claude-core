# ブランチ作成ルール

プレフィックス対応表・命名ルール・AI判断ガイドラインは `~/.claude/skills/git-workflow/SKILL.md` を参照すること。

## 実行手順

1. 現在のブランチを確認
2. developブランチに切り替え
3. developブランチを最新化（git pull origin develop）
4. 新しいブランチを作成して切り替え
5. 結果を表示

⚠️ **注意**: ブランチ作成・切り替え後にはいかなる作業も行いません。必要な作業は別途手動で実行してください。
